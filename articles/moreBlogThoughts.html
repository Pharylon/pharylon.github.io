More Blog Thoughts

Nov 14, 2018

I'm going to set down a few ideas about what I want to accomplish with my as-yet-unnamed blogging framework. What are my goals and what do I hope to accomplish with it? Here are a few preliminary thoughts.

  • It uses vanilla HTML, CSS, and Javascript, so it should run forever.
  • No build tools. The source is the site and the site is the source.
  • It should be approachable for anyone that knows HTML.
  • It should be easily understood and customizable to anyone that knows Javascript.
  • No feature creep. It does the bare minimum of what a blog needs to do.

I suppose that covers it for now. I'm debating what my next thing should be. Maybe tags? Those seem like something most blogs have, but does that go against "feature creep" and "doing the bare minimum?". A mobile view would be nice, though it looks pretty OK on phones already.
